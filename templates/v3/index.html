Instrucciones

Por favor, completa todos los campos requeridos para generar el PDF de phishing personalizado. Asegúrate de ingresar datos válidos y precisos:

  • Nombre del invitado: Ingrese el nombre completo del invitado.
  • Fecha y hora del evento: Especifique la fecha y hora del evento.
  • Lugar: Proporcione el lugar del evento.
  • Ubicación: Incluya un enlace de ubicación o una descripción detallada.
  • Imagen: Añada un enlace a la imagen decorativa para el PDF.
